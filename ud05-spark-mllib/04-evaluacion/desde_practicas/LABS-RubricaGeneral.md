# Rúbrica general — Laboratorios de Spark MLlib

```markdown
# Rúbrica de evaluación
Laboratorios de Machine Learning con Spark

Esta rúbrica se utiliza para evaluar los laboratorios de Machine Learning realizados con Spark MLlib.

El objetivo es valorar tanto la correcta implementación técnica como la comprensión del proceso de Machine Learning.

---

# Criterios de evaluación

| Criterio | Excelente | Correcto | Básico | Insuficiente | Puntos |
|--------|-----------|---------|--------|-------------|-------|
| Carga y preparación de datos | Dataset cargado correctamente, exploración clara y estructura bien comprendida | Dataset cargado correctamente | Dataset cargado pero con errores menores | Dataset incorrecto o no funcional | 2 |
| Preparación de features | Uso correcto de VectorAssembler y selección adecuada de variables | Features creadas correctamente | Features incompletas o poco claras | Features incorrectas o inexistentes | 2 |
| Entrenamiento del modelo | Modelo entrenado correctamente con configuración clara | Modelo entrenado correctamente | Modelo entrenado con errores menores | Modelo no funcional | 2 |
| Predicciones | Predicciones generadas correctamente y mostradas con claridad | Predicciones correctas | Predicciones incompletas | No hay predicciones | 2 |
| Evaluación e interpretación | Métricas calculadas correctamente y breve interpretación del resultado | Métrica calculada correctamente | Métrica calculada sin interpretación | No se evalúa el modelo | 2 |

---

# Puntuación total

Puntuación máxima: 10 puntos

Interpretación orientativa:

| Puntuación | Nivel |
|-----------|------|
| 9 – 10 | Excelente |
| 7 – 8 | Muy buen trabajo |
| 5 – 6 | Aprobado |
| < 5 | Insuficiente |

---

# Aspectos adicionales que se valoran

También se tendrá en cuenta:

- claridad del notebook
- organización del código
- comentarios explicativos
- ejecución completa sin errores

Un notebook que no se ejecuta completamente puede ver reducida su calificación.

---

# Recomendaciones para los estudiantes

Antes de entregar el laboratorio:

1. Ejecutar todas las celdas del notebook desde el principio.
2. Verificar que no aparecen errores.
3. Comprobar que se muestran resultados intermedios.
4. Revisar las respuestas a las preguntas de reflexión.
```


# *Penalización estándar para todos los labs**:

```
Notebook que no ejecuta completo: -2 puntos
```
