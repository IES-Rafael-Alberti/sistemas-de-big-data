# UD1 — Introducción a Big Data

Esta unidad introduce el problema Big Data, la calidad de datos y las primeras decisiones de arquitectura. El objetivo no es memorizar herramientas, sino entender cuándo un sistema de datos necesita otro enfoque.

## Ruta de trabajo

1. Empieza por [Big Data 101](../ud01-introduccion-big-data/01-teoria/UD1_01_BigData101.md).
2. Continúa con [EDA y calidad de datos](../ud01-introduccion-big-data/01-teoria/UD1_02_EDA_y_calidad.md).
3. Revisa la base de [estadística para Big Data](../ud01-introduccion-big-data/01-teoria/UD1-Parte0-BaseMatematica/UD1_P0_Estadistica_para_BigData.md).
4. Practica con el [laboratorio de logs JSON](../ud01-introduccion-big-data/03-practicas/UD1-Parte4/lab_logs_json/UD1_Lab_Logs_JSON.md).
5. Cierra con la [tarea y rúbrica](../ud01-introduccion-big-data/04-evaluacion/UD1_03_Tarea_y_rubrica.md).
