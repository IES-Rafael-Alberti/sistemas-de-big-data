# UD5 — Spark MLlib

Esta unidad introduce aprendizaje automático distribuido con Spark MLlib. El objetivo es construir pipelines de datos y modelos básicos que puedan trabajar con volúmenes grandes.

## Ruta de trabajo

1. Lee [introducción a ML en Big Data](../ud05-spark-mllib/01-teoria/UD5_01_intro_ml_bigdata.md).
2. Continúa con [pipeline de datos para Big Data](../ud05-spark-mllib/01-teoria/UD5_02_pipeline_datos_bigdata.md).
3. Revisa [Spark MLlib](../ud05-spark-mllib/01-teoria/UD5_03_spark_mllib.md).
4. Practica con [regresión distribuida](../ud05-spark-mllib/03-practicas/LAB1-RegresionDistribuidaConSparkMLlib.md).
5. Cierra con [clasificación con Titanic](../ud05-spark-mllib/03-practicas/LAB3-ClasificacionConSparkMLlib-Titanic.md).
