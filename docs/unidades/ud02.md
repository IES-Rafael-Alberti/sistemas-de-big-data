# UD2 — Almacenamiento e ingesta

Esta unidad trabaja la entrada de datos en sistemas Big Data: fuentes, formatos, almacenamiento, calidad incremental y primeras decisiones de seguridad.

## Ruta de trabajo

1. Lee [problema y arquitectura](../ud02-almacenamiento-ingesta/01-teoria/UD2_01_Problema_y_arquitectura.md).
2. Continúa con [ingesta con scripts y Airbyte](../ud02-almacenamiento-ingesta/01-teoria/UD2_02_Ingesta_scripts_y_Airbyte.md).
3. Revisa [integración y calidad incremental](../ud02-almacenamiento-ingesta/01-teoria/UD2_03_Integracion_y_calidad_incremental.md).
4. Practica con [Medallion local, Parquet, DuckDB y Spark](../ud02-almacenamiento-ingesta/03-practicas/UD2_Practica_Local_Medallion_Parquet_DuckDB_Spark.md).
5. Entrega la [tarea y rúbrica de UD2](../ud02-almacenamiento-ingesta/04-evaluacion/UD2_05_Tarea_y_rubrica.md).
