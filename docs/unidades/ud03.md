# UD3 — Procesamiento distribuido

Esta unidad introduce Spark y el procesamiento distribuido. El foco está en entender cuándo conviene distribuir un cálculo, cómo medir rendimiento y cómo preparar datos para análisis posterior.

## Ruta de trabajo

1. Lee [procesamiento distribuido y analítica escalable](../ud03-procesamiento-distribuido/01-teoria/UD3-ProcesamientoDistribuidoAnaliticaEscalable.md).
2. Haz [SparkLab 1](../ud03-procesamiento-distribuido/03-practicas/Spark_Labs/SparkLab1/01-SparkLab1.md).
3. Continúa con [SparkLab 2](../ud03-procesamiento-distribuido/03-practicas/Spark_Labs/SparkLab2/02-SparkLab2.md).
4. Trabaja [Spark, Parquet y particionado](../ud03-procesamiento-distribuido/03-practicas/Spark_Labs/SparkLab3/00-SparkLab3.md).
5. Cierra con [streaming con Spark y Redpanda](../ud03-procesamiento-distribuido/03-practicas/Streaming_Labs/SparkLab5-Streaming/01-SparkLab5-Streaming.md).
