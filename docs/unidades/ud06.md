# UD6 — Proyecto integrador

Esta unidad organiza el proyecto final. El producto esperado es un sistema de datos reproducible que conecte ingesta, calidad, procesamiento, visualización y documentación técnica.

## Ruta de trabajo

1. Lee el [guion del proyecto](../ud06-proyecto/guion_proyecto.md).
2. Elige un dominio con al menos dos fuentes de datos.
3. Diseña la arquitectura Medallion del proyecto.
4. Construye el pipeline reproducible desde Bronze hasta Gold.
5. Documenta decisiones, calidad, RGPD y uso de IA.
