# Sistemas de Big Data

Material didáctico del módulo **Sistemas de Big Data** — Curso 2026/2027, IES Rafael Alberti.

Usa esta página como punto de entrada. Cada unidad tiene una guía propia con la ruta recomendada, materiales clave y enlaces directos a teoría, prácticas y entregas.

| Unidad | Descripción |
|--------|-------------|

| [UD2 — Almacenamiento e ingesta](unidades/ud02.md) | HDFS, NoSQL, ingesta, calidad, RGPD, Medallion |
| [UD3 — Procesamiento distribuido](unidades/ud03.md) | Spark, Hadoop, streaming, Kibana, Grafana |
| [UD4 — BI y orquestación](unidades/ud04.md) | Metabase, Superset, Airflow, dashboard técnico |
| [UD5 — Spark MLlib](unidades/ud05.md) | ML pipelines, scikit-learn vs Spark, regresión y clasificación |
| [UD6 — Proyecto integrador](unidades/ud06.md) | Proyecto multi-módulo SBD + BDA + PIA |
