# 📊 Rúbrica de Evaluación 

## 🧾 Evaluación global (sobre 10)

| Criterio              | Descripción                            | Puntos |
| --------------------- | -------------------------------------- | ------ |
| Pipeline funcional    | Ejecuta correctamente sin errores      | 2      |
| Data Loader           | Carga correcta de datos                | 1      |
| Transformación        | Lógica aplicada correctamente          | 2      |
| Exportación           | Resultado generado correctamente       | 1      |
| Uso de pandas         | Correcto y coherente                   | 1      |
| Estructura y claridad | Código limpio y bloques bien definidos | 1      |
| Documento             | Explicación clara y correcta           | 1      |
| Evidencias            | Capturas completas                     | 1      |

---

## 🧠 Evaluación cualitativa (rápida para corrección)

| Nivel          | Descripción                                        |
| -------------- | -------------------------------------------------- |
| ⭐ Excelente   | Pipeline completo, bien pensado, con valor añadido |
| 👍 Muy bueno   | Todo correcto, bien explicado                      |
| ✔ Adecuado     | Funciona pero básico                               |
| ⚠ Mejorable    | Errores o falta de claridad                        |
| ❌ Insuficiente| No funcional o incompleto                          |

---

## 🧾 Formato de corrección (para ti, rápido)

Ejemplo:

```id="3pt4b2"
Pipeline → OK  
Transformación → Mejora (poco relevante)  
Documento → OK  
Evidencias → OK  

Nota: 7.5
```
