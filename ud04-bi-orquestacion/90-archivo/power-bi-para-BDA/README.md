# Power BI — Material archivado para Big Data Aplicado

Este directorio contiene el material de Power BI que **se ha retirado de la ruta
principal de Sistemas de Big Data (SBD)** porque su enfoque es BI de negocio,
no visualización técnica.

Según el criterio curricular acordado en la reorganización:

> **SBD** → visualización técnica mínima (Metabase, Superset, dashboard técnico)
> **BDA** → BI orientada a negocio, dashboards de solución final, Power BI

Si impartes **Big Data Aplicado**, este material puede serte útil tal cual o como
base para adaptar.

En SBD solo se conserva la **comparativa conceptual** de herramientas BI:

## Notas para el profesor de BDA

- Power BI Desktop requiere **Windows**. En Linux se puede usar la versión web
  (Power BI Service) con limitaciones.
- La licencia gratuita de Power BI Desktop permite crear informes, pero
  **compartirlos** requiere licencia Power BI Pro (≈10€/usuario/mes).
- Si el centro tiene convenio Microsoft o licencias educativas, se puede
  obtener Power BI Pro gratuito para profesorado.
- El dataset `supermarket_Sales.csv` es un ejemplo clásico de BI de negocio
  (ventas por sucursal, línea de producto, género, etc.) y puede servir como
  base para prácticas más profundas en BDA.
